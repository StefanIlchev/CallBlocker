# CallBlocker

A call blocker for [Android](https://www.android.com/) that rejects calls from numbers that match or do not match a given [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

## Install

### Sideload

Download and install the `.apk` file of the [latest release](https://github.com/StefanIlchev/CallBlocker/releases/latest).

### Google Play (Early Access)

Join the tester [Google Group](https://groups.google.com/g/call-blocker) and use the link from the welcome message to become a tester. After that the app will be visible on the [Play Store](https://play.google.com/store/apps/details?id=stef40.callblocker).
